# European Football Players Dataset 2025

## Overview

This project contains a manually compiled football dataset featuring 134 players from five major European clubs:

- Barcelona
- Real Madrid
- Liverpool
- Arsenal
- Manchester City

## Dataset Files

### Football.xlsx
Contains player information including:
- Player_ID
- Player_Name
- Club
- League
- Position
- Date_of_Birth
- Nationality
- Height_cm

### clubs.xlsx
Contains club information including:
- Club_ID
- Club
- League
- Country

### football_eda.ipynb
Exploratory Data Analysis performed using Python, Pandas, and Matplotlib.

## Dataset Statistics

- Total Players: 134
- Total Clubs: 5
- Leagues Represented:
  - Premier League
  - La Liga
